# chakula-v2

An AI powered recipe generator application that is useful for creating personalised recipes based on caloric needs, available pantry items and dietary preferences. Recipes generated by any user are saved to a global collection that is accessible by everyone

**Status:** Under construction

## Background

This is a rebuild of an earlier project ![Chakula v1](https://github.com/joeehis1/chakula-v1-archive) that bundled AI recipe
generation with grocery price scraping. In rebuilding, I cut the scraper to
focus on what the app does well: personalized AI-generated recipes grounded
in what the user actually has on hand.

## Stack

- **Framework:** Next JS 16 using App router
- **Styling:** Tailwind CSS
- **Database:** PostgreSQL (Neon) with Prisma ORM
- **Auth:** Auth.js (NextAuth)
- **AI:** Groq (Llama 3.3 70B) via the Vercel AI SDK for streaming
- **Validation:** Zod
- **Testing:** Jest + React Testing Library
- **CI:** GitHub Actions
- **Deploy:** Vercel
